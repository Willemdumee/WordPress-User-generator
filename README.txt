Easily generate a bunch of new - fake - users to test your plugin / theme of website

== Description ==

Generate fake users easily for testing purposes. You can choose between 3 themes:
* Star Wars
* The Simpsons
* WordPress guru's

User Generator will generate id, username, screenname and email for the amount of users that you have set.


== Installation ==

1. Upload `ctrl-user-generator` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
